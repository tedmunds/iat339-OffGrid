Welcome to OffGrid!

Off Grid offers a way for anyone with the will, to start living off the grid. This mean sthat you can generate your own electricity, 100% renewably and environmentally friendly. It can be a bit complicated, so this site will help you get started, letting you know everything you will need and how to use it.
